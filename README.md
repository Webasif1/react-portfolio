
# React Navbar UI (CDN-Based)

A responsive navigation bar built using **React via CDN** without JSX or build tools.  
This project focuses on understanding **core React fundamentals** such as `React.createElement`, component-based structure, and state management using `useState`.

This is a beginner-friendly project aimed at strengthening React basics before moving to modern tooling.
